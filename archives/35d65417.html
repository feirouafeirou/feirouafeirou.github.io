API 与 WebAPI

  • API：应用程序编程接口 (实现某种功能的接口)
  • WebAPI：操作浏览器功能的接口 (BOM，DOM)

DOM

  • DOM 是一种接口
  • DOM 把以下这些都看作对象：
    • 文档(document)：整个页面
    • 元素(element)：标签
    • 节点(node)：标签，属性，文本，注释
